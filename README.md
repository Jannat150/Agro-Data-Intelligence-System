# Agro-Data-Intelligence-System

## 📌 Overview
This project analyzes agricultural data to understand how **weather conditions** and **production factors** impact crop yield and farming risks.  

Using data science techniques, the project helps in:
- Identifying climate risks 
- Understanding crop production trends 
- Generating insights for better farming decisions 

---

## 🎯 Objectives
- Analyze weather patterns and detect risky conditions  
- Study crop production trends over time  
- Understand the relationship between climate and agriculture  
- Generate actionable insights for farmers  

---

## 🛠️ Tech Stack
- **Python**
- **NumPy** – numerical computations  
- **Pandas** – data cleaning & manipulation  
- **Matplotlib** – data visualization  
- **Seaborn** – advanced visualization  

---

## 📂 Datasets Used
1. **Weather Dataset**
   - Temperature, pressure, radiation, wind, etc.
   - Used for climate risk analysis  

2. **Crop Production Dataset**
   - Area, production, yield (Rice, Wheat, etc.)
   - Used for supply and yield analysis  

3. **State Dataset**
   - State-level metadata
   - Used for basic geographical insights  

---

## 🔄 Project Workflow

### 1️⃣ Data Cleaning
- Handled missing values  
- Standardized column names  
- Removed inconsistent data  

### 2️⃣ Exploratory Data Analysis (EDA)
- Temperature distribution  
- Radiation vs risk  
- Production trends over years  
- Area vs yield comparison  

### 3️⃣ Visualization
- Line plots for production trends  
- Scatter plots for relationships  
- Heatmaps for correlation  
- Count plots for risk distribution  

### 4️⃣ Insight Generation
- Climate conditions affect crop stability  
- Production varies significantly across years  
- High temperature & radiation increase risk  
- Larger farming area does not always mean higher yield  

---

## 📊 Key Insights
-  Higher temperatures are linked with increased climate risk  
-  Weather variability impacts agricultural productivity  
-  Crop production trends fluctuate over time  
-  Climate risk plays a crucial role in farming outcomes
